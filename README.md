# Comandos SQL

## Comandos Basicos de SQL

### Crear Base de Datos
Crea una nueva base de datos.
```
CREATE DATABASE ########;
```

### Usar Base de Datos
Selecciona la base de datos para el siguiente comando.
```
USE ########;
```

### Borrar una Base de Datos
Borra una base de datos al completo.
```
DROP DATABASE ########;
```
